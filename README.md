# Decentralized Guardian

Open-source AI + blockchain system to detect corruption in rural communities.

## How It Works
1. Sensors collect water/fund data.
2. AI agent detects anomalies.
3. Blockchain logs permanently.
4. Dashboard shows live alerts.

## Tech Stack
- Python 3.9+
- TensorFlow Lite
- Ethereum (Sepolia)
- Raspberry Pi
- React
